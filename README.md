# 📚 LibraTech

Sistema web para gerenciamento de bibliotecas desenvolvido em **ASP.NET Core MVC**, permitindo o controle de livros, usuários, categorias e empréstimos por meio de uma interface moderna, organizada e responsiva.

---

## 📖 Sobre o projeto

O **LibraTech** foi desenvolvido como projeto acadêmico da disciplina de Desenvolvimento Web, aplicando o padrão arquitetural **MVC (Model-View-Controller)** e boas práticas de desenvolvimento utilizando a plataforma .NET.
O sistema permite administrar o acervo de uma biblioteca de forma simples e eficiente, centralizando o gerenciamento de livros, usuários, categorias e empréstimos em uma única aplicação.

---

## ✨ Funcionalidades

### 📊 Dashboard
- Indicadores em tempo real
- Total de livros cadastrados
- Total de usuários
- Empréstimos ativos
- Livros disponíveis
- Gráfico de livros por categoria

### 📚 Livros
- Cadastro de livros
- Edição e exclusão
- Consulta de informações
- Controle de ISBN
- Cadastro de editora
- Quantidade disponível
- Sinopse
- Associação com categorias

### 👤 Usuários
- Cadastro de usuários
- Alteração de dados
- Exclusão
- Controle de CPF
- Controle de telefone
- Cadastro de e-mail

### 📂 Categorias
- Cadastro
- Edição
- Exclusão
- Organização do acervo

### 📖 Empréstimos
- Registro de empréstimos
- Controle de devoluções
- Consulta da situação dos empréstimos

### ✉️ Contato
- Página institucional de contato
- Informações da empresa
- Formulário de contato

## 🎯 Objetivo

Este projeto foi desenvolvido para aplicar os principais conceitos de desenvolvimento web utilizando ASP.NET Core MVC, incluindo:

- Arquitetura MVC
- CRUD completo
- Entity Framework Core
- Relacionamentos entre entidades
- ViewModels
- Validação de formulários
- Dashboard administrativo
- Interface responsiva
